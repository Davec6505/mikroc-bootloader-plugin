# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please:

1. **Do NOT** open a public issue
2. Email security concerns to: [davec6505@gmail.com] (update this)
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

We will respond within 48 hours and work with you to address the issue.

## Security Considerations

This extension executes external commands (mikro_hb.exe). Users should:
- Only use bootloader executables from trusted sources
- Verify file paths in settings
- Be cautious with workspace .hex files
